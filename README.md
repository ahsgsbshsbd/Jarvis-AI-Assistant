# Jarvis Assistant

A desktop AI assistant written in Java.

Features

- GUI chat interface
- Voice input
- Speech output
- Memory system
- Internet tools
- Jarvis-style assistant

Build Instructions

1 Install Java JDK 17+
2 Install Maven
3 Double-click build.bat

Output

dist/JarvisAssistant.jar

Run

java -jar dist/JarvisAssistant.jar
